For Professional version subscribers, the next federal program that HHLEARN can assist your organization with is checking the List of Excluded Individuals and Entities (LEIE) database. This is a list of individuals and organizations that have been excluded from working for any company that bills Medicare. We check this list every night for the individuals that were added to the system that day and once every quarter. We then provide you with a report that can be used to prove your compliance with this regulation (You can even run this report at any time if you have the Professional version.).

If you opt to utilize this feature, it requires that you provide a valid social security number for each employee placed on the system. Social security numbers, which we encrypt with 256 AES (advanced encryption standard) and hash and salt them, are the only way to validate that a person with a specific name is not on the list. (Any other site that is providing this service without valid social security numbers is providing your organization with incorrect and misleading information.)
