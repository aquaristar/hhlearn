EMPLOYEE HANDBOOK

HHLEARN's Employee Handbook Module (EHM) is designed to produce a customized employee handbook for each organization. Our EHM asks a series of questions and stores your specific data. The EHM then uses the stored data to prepare a custom employee handbook in Microsoft Word or Portable Document Format (PDF). What makes HHLEARN's Employee Handbook Module so unique is that a specific version is created for each state-specific location, so if you have locations in different states, a concise version will be created for each location and not one bloated version. HHLEARN also stores every version of your employee handbook so your organization has an audit trail.

INSERVICE TRACKER

Every organization provides inservice education, and keeping track of this information is difficult. Our Inservice Tracking Module (ITM) allows you to automatically select the employees that attend each inservice and upload supporting documents that pertain to the inservice, e.g., ; files, ; files, PDFs, etc. These files then can be downloaded when needed to support the education topic. In addition, our ITM stores the attended inservices so they can be seen along with each employee's HHLEARN education history. Organizations can also print out professionally designed inservice rosters to be printed for hardcopy records (the non-green approach).

EVENT TRACKER

Our Event Tracking Module (ETM) allows organizations to setup events that the organization needs to be alerted about. The system comes with a preloaded list of events, but each organization can load their own events and have email or text (SMS) messages sent to any employee. These events can be setup as one-time events or recurring at almost any frequency. All events are stored and the acknowledgments of these events are also stored for audit purposes.
